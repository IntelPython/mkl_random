mkl_random APIs#

The class mkl_random.RandomState exposes sampling from probability distributions while supporting different streams of randomness, also known as basic random number generators.

The basic random number generator is chosen by specifying brng keyword argument to the constructor of mkl.RandomState class.

The list of supported basic random number generators is as follows (also see oneMKL Engines):

  • 'MT19937' - the Mersenne Twister pseudo-random number generator (default), example

  • 'SFMT19937' - the SIMD-oriented Mersenne Twister pseudo-random number generator, example

  • 'MT2203' - the set of 6024 Mersenne Twister pseudorandom number generators, example

  • 'R250' - the 32-bit generalized feedback shift register pseudorandom number generator GFSR(250,103), example

  • 'WH' - the set of 273 Wichmann-Hill’s combined multiplicative congruential generators, example

  • 'MCG31' - the 31-bit multiplicative congruential pseudorandom number generator, example

  • 'MCG59' - the 59-bit multiplicative congruential pseudorandom number generator, example

  • 'MRG32K3A' - the combined multiple recursive pseudorandom number generator MRG32k3a, example

  • 'PHILOX4X32X10' - the Philox4x32x10 counter-based pseudorandom number generator, example

  • 'NONDETERM' - the generator with non-deterministic source of randomness (for example, a hardware device), example

  • 'ARS5' - the ARS5 counter-based pseudorandom number generator based on AES encryption algorithm, example
